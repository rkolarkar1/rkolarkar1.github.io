# Rahul Kolarkar Portfolio

GitHub Pages portfolio website.

## Portfolio URL

After publishing, the site will be available at:

```text
https://rkolarkar1.github.io
```

## Files

```text
index.html
style.css
README.md
```

## How to publish

1. Create a public GitHub repo named exactly:

```text
rkolarkar1.github.io
```

2. Upload `index.html`, `style.css`, and `README.md`.
3. Commit to the `main` branch.
4. Open `https://rkolarkar1.github.io` after a few minutes.

## Check before submitting

If your Project 1 repository name is different, update the link in `index.html`.
